Guix中国镜像上线啦

经过一个月的筹备，Guix中国镜像上线啦。我们购买了 guix.org.cn 域名和一个服务器，并完成了域名备案。此服务器用于提供Guix软件镜像服务。

服务器配置如下：

  • 处理器：2核
  • 内存：4GB
  • 硬盘：50GB
  • 上行带宽：5Mbps

服务器的软件配置可在 GitHub 查看。

Guix镜像服务的地址是 https://mirror.guix.org.cn/ 。欢迎测试。
